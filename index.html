Slack API Dummy Host

Static, GitHub Pages-friendly Slack API simulator with dummy responses. Includes an interactive explorer with prefilled parameters and one-click execution.

How this works on GitHub Pages

  • No Python server required: this project is 100% HTML/CSS/JavaScript.
  • API calls are simulated in-browser via a deterministic mock router.
  • You can call the simulator programmatically using window.slackMockApi.handleRequest(...).
Programmatic usage example 
const result = await window.slackMockApi.handleRequest({
  method: 'POST',
  path: '/api/slack/chat/postMessage',
  params: { channel: 'C123', text: 'hello' }
});
console.log(result);
